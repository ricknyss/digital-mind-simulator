# Digital Mind Simulator

Digital Mind Simulator is a Python-based terminal program that simulates a virtual mind with emotional and physical states.

The system tracks multiple attributes including:

- Energy
- Mood
- Stress
- Hunger

Users can interact with the system by performing actions that affect the mind's state.

## Features

- Interactive command line simulation
- Dynamic state updates
- Decision based interaction
- Modular Python code structure

## Technologies

Python

## How to Run

Clone the repository:

git clone https://github.com/ricknyss/digital-mind-simulator.git

Run the program:

python main.py
